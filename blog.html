Is it really possible to get a WiFi/Bluetooth hacking tool on such a low budget?
In fact, it actually is!

All you need is to get an ESP32 module and install the "ESP32-Marauder" firmware from JustCallMeKoko.
(a how-to guide will be provided later)

Currently, the fastest, most affordable, and effective way to get a module that includes everything we need for this project is to grab an ESP32-2432S028 development board (better known as the "Cheap Yellow Display"). It’s commonly available on Amazon, AliExpress, and other platforms with prices around 3€ and 10€ (may vary).

This board comes with a built-in touchscreen, a Micro-SD card slot, and both Micro-USB and USB Type-C power ports.

What is the ESP32-Marauder firmware by JustCallMeKoko?

The ESP32-Marauder firmware, developed by JustCallMeKoko, is a tool specifically designed to fully leverage the capabilities of the ESP32 module in the field of security and hacking. This firmware allows users to perform a variety of attacks and penetration tests on WiFi and Bluetooth networks, transforming the ESP32 into a multifunctional device for network analysis. With features such as network scanning, data interception, and connection manipulation, it has become a popular choice among cybersecurity enthusiasts and ethical hacking professionals.

Advantages of Using the Web Flasher by Fr4nkFletcher

One of the main advantages of installing the ESP32 Marauder firmware is the ease provided by the web flasher created by Fr4nkFletcher. This flasher offers an intuitive and accessible interface that allows users to quickly and effortlessly load the firmware onto their ESP32 module. With its streamlined design, even those with limited technical experience can carry out the installation without complicated procedures. Furthermore, the web flasher eliminates the need to set up development tools, further facilitating access to this powerful hacking tool. In summary, the ESP32 Marauder, combined with Fr4nkFletcher's flasher, makes the installation and configuration process accessible to everyone.

Is it open source?

Yes, this project is open source. You can download "ESP32-Marauder" firmware from JustCallMeKoko. and load it as a project in the Arduino IDE to configure your own version of the firmware

On the other hand, if you choose to use the web flasher, it will upload the latest available version of the firmware to your device without allowing you the opportunity to modify the code.
